
## Cookie如何获取
获取自己的Cookie信息

通过浏览器登录米哈游论坛 https://bbs.mihoyo.com/ys/

按F12，打开开发者工具 -> Console

```
confirm("确认将cookie复制到剪切板中？")&&copy(document.cookie)
```
